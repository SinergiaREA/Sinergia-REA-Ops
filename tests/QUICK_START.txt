## ⚡ 30 SEGUNDOS — Verificación Rápida

1. Abre la app e inicia sesión
2. Presiona **F12** (abre consola)
3. Escribe y presiona ENTER:
   ```

   ```

## ❌ PROBLEMAS COMUNES

### "FCM: PERMISSION_DENIED"
El navegador bloqueó las notificaciones.
**Solución:** Configuración → Sinergia REA → Notificaciones → "Permitir"

### "FCM: TOKEN_ERROR"
Hay error en la comunicación con Firebase.
**Solución:**
1. Asegúrate de estar en **HTTPS** (no HTTP)
2. Recarga la página con **Ctrl+F5**
3. Verifica que Firebase Console tenga **API FCM V1 habilitada**

### "No hay usuario autenticado"
No iniciaste sesión correctamente.
**Solución:** Inicia sesión con una cuenta autorizada

### No me aparece el botón "🔍 FCM"
No eres usuario admin.
**Solución:** El botón solo aparece para admins. Si eres admin, ejecuta:
```
window.__FCM_Tests.status()
```

```javascript
// Ver logs detallados
window.__FCM_Tests.logs()

// Ver usuario autenticado
window.__FCM_Tests.user()

// Verificar token en Firestore
window.__FCM_Tests.firestore()
```

## 📱 ENVIAR NOTIFICACIÓN DE PRUEBA

1. Firebase Console → Cloud Messaging
2. "Enviar tu primer mensaje"
3. Llenar:
   - Título: "Test FCM"
   - Cuerpo: "¡Funciona!"
   - Seleccionar por token o UID
4. "Enviar"

Deberías recibir la notificación de inmediato.
